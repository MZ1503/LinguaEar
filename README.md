# 🎤 LinguaEar

### Your AI German Learning Companion

LinguaEar listens to German conversations around you and explains them in real time — built for immigrants and expats in Germany.

## Features

- 🎤 Real time German speech recognition
- 📝 Instant transcription
- 💡 Contextual explanation in English
- 😊 Tone and sentiment analysis
- 🔄 Reset button
- 🔒 Privacy Policy & Impressum

## Tech Stack

- React Native + Expo
- OpenAI Whisper — Speech recognition
- GPT-4o-mini — Language explanation
- EAS — Build and deployment

Built by an immigrant living in Germany who couldn't understand anyone around her. LinguaEar was validated at AI Tinkerers Cologne.
