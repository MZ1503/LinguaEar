# 🎤 LinguaEar

### Your AI German Learning Companion

LinguaEar listens to German conversations around you and explains them in real time — built for immigrants and expats in Germany. I recently found that LinguaEar can also explain the English phrase in German. It works both ways. Which I thought to be a bug is an additional feature of the app.

## Features

- 🎤 Real time German/English speech recognition
- 📝 Instant transcription
- 💡 Contextual explanation in English/German
- 😊 Tone and sentiment analysis
- 🔄 Reset button
- 🔒 Privacy Policy & Impressum

## Tech Stack

- React Native + Expo
- OpenAI Whisper — Speech recognition
- GPT-4o-mini — Language explanation
- EAS — Build and deployment
